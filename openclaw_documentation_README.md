# opf

OpenPecha Format specifications

## Description

OPF is an open-directory file format containing a text and its annotations in an easy to parse, metadata and optionally the source of the text.

Metadata structure in OpenPecha is based on the BibFrame2 model, while annotations follow the TEI standard as much as possible.

There are two types of .opf documents: Work files and Pecha files. Works are BibFrame2 works and Pechas correspond to BibFrame2 Instances. Thus a Work can have multiple Pechas. Exact digital duplicates are recorded as Items in the Work metadata.
